Intelligent Threat Prevention System (ITPS)

An AI-driven intrusion prevention system combining anomaly and signature-based detection for web application threats.

Overview

ITPS processes HTTP traffic and evaluates it using a hybrid model to stop SQL Injection, XSS, CSRF and other attacks before they reach the application. The system continuously learns from new traffic patterns to improve detection accuracy.

Key Features

Machine learning anomaly detection
Signature matching for known attacks
Real-time mitigation
Continual model training

Technology Stack

  • Language: Python
  • Framework: FastAPI
  • Libraries: Scikit-learn, TensorFlow
