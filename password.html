Secure Password Management System

Credential vault built with FastAPI and Python to demonstrate secure software development practices.

Overview

The system encrypts passwords client-side and stores them in a server-side vault protected by hashing and salting techniques. Role-based access and audit logs ensure that only authorized users can retrieve secrets.

Key Features

Encrypted storage of credentials
Role-based access controls
Audit logging for every operation
Secure password generation and rotation

Technology Stack

  • Framework: FastAPI
  • Language: Python
  • Database: SQLite with SQLAlchemy
